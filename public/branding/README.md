# Branding Assets

This folder contains the logo assets for YO! TECH THIS OUT.

## Required Files

Place the following PNG image files in this directory:

- **logo-dark.png** - Full logo for dark theme (used in navigation, loading intro, footer)
- **logo-light.png** - Full logo for light theme (used in navigation, loading intro, footer)
- **logo-mark.png** - Compact logo mark (used in mobile navigation)

## File Specifications

### Full Logos (logo-dark.png, logo-light.png)
- Recommended dimensions: 500x150px (or proportional)
- Format: PNG with transparency
- Use case: Navigation bar (desktop), loading intro, footer

### Compact Mark (logo-mark.png)
- Recommended dimensions: 48x48px (or proportional)
- Format: PNG with transparency
- Use case: Navigation bar (mobile/small screens)

## Notes

- The logo component will automatically fall back to text if images are missing
- Images should have transparent backgrounds
- Ensure proper contrast for both dark and light themes
- The component handles theme switching automatically
